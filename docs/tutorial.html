サイドバー機能

Emacsライクな操作を視覚化するサイドバースイート

ファイラ — C-x d

ファイルエクスプローラ。l/Right でプレビュー。r でドライブルートへ。

アウトライン — C-c o

Markdown・HTMLの見出し階層をリアルタイム表示。最大50,000行対応。

Kill Ring — C-c y

削除・コピーの履歴を一覧。選択してヤンク可能。最大60件。

レジスタ — C-c r

テキストや位置を永続保存（ローカルストレージ）。セッション間も保持。

最近使ったファイル — C-c c

最近開いたファイルを一覧。Space でプレビュー。パス表示付き。

ワークスペース — C-.

複数フォルダをまとめて管理。VS Code互換の .code-workspace に対応。

MCP サーバー設定

Claude・Google Antigravity と elecxzy を連携させる

🤖elecxzy は MCPサーバーとして動作します。AIが未保存の編集状態を含むバッファを直接読み書きできます。
claude_desktop_config.json
{ "mcpServers": { "elecxzy": { "command": "C:\\Users\\[ユーザー名]\\AppData\\Local\\Programs\\elecxzy\\elecxzy.exe", "args": [ "C:\\Users\\[ユーザー名]\\AppData\\Local\\Programs\\elecxzy\\resources\\app.asar" ], "env": { "ELECXZY_MCP_MODE": "true", "ELECTRON_RUN_AS_NODE": "1" } } } }
⚠️[ユーザー名] を実際のWindowsユーザー名に置き換えてください。Microsoft Store版と GitHub版でインストールパスが異なる場合があります。
起動コマンド

設定後、elecxzy を起動してから M-x mcp-start または C-c m でMCPサーバーを開始します。

FAQ

右端での折り返し表示はできますか？

現時点では動的な折り返し（ソフトラップ）には対応していません。M-q（fill-paragraph）によるハードラップ、または M-x auto-fill-mode による自動改行をご利用ください。

チルトホイールで水平スクロールできません。

Logi Options などのマウスユーティリティが水平スクロール信号をブロックしている場合があります。Shift + ホイール回転で水平スクロールが可能です。

MS ゴシックでカーソル位置がずれます。

古いフォントのビットマップ仕様と Canvas API の文字幅計測の競合です。BIZ UDGothic などのモダンな等幅フォントを推奨します。
