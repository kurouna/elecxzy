Emacs-Inspired Experience

手に馴染んだキーバインドを、
そのまま最新の環境へ。

C-f/b/n/p による基本移動から、Kill Ring、マークセット、ウィンドウ分割まで。長年培われてきた直感的な操作体系を、設定不要でそのまま利用できます。

ブラウザ世代の技術スタックにより、高速なレスポンスと高い拡張性を両立しました。

日本語対応

標準でのIMEサポート。

C-\ による IME ON/OFF 切り替えはもちろん、M-x や C-u などのプレフィックスキー入力時には自動的に IME をオフにするなど、デフォルト状態での使いやすさにこだわっています。

また、Intl.Segmenter を活用した日本語処理により、M-f/M-b での文節単位の移動や、ポップアップ時の正確な折り返しにも対応しています。

Piece Table Engine

高速・省メモリな
バッファ管理。

RB-Tree ベースの Piece Table アルゴリズムを採用。大容量ファイルでもメモリ効率よく動作し、インクリメンタルなUndo履歴を実現します。

Quick Navigation

Avy ジャンプで高速移動。

C-' に続けて文字を入力し、画面内の任意の場所へ数ストロークでジャンプ。視覚的なラベルにより、マウスを使わず正確な移動が可能です。

また、インクリメンタル検索（C-s / C-r）の実行中にも C-' を押すことで、検索結果を対象とした Avy ジャンプへシームレスに移行できます。

Visual Effects

打鍵を彩るカーソル VFX。

Light Particle や Cyber Trace など、9種類の視覚エフェクトを搭載。タイピングの体験をより楽しく演出します。

All Features

主な機能

Emacsライクな操作体系を継承しつつ、現代の開発体験に合わせた機能セットです。

⌨

Emacsライク キーバインド

C-f/b/n/p の基本移動からKill Ring、ウィンドウ分割まで対応しています。M-x で全コマンドをインクリメンタル補完付きで実行できます。

▦

矩形編集

列単位の一括編集を行います。C-x SPC で矩形マークモード、C-x r k/y/t でkill・yank・文字列置換が可能です。専用のKill Ringも備えています。

🏃

Avy ジャンプ

C-' に続けて文字を入力すると、画面内の一致箇所にラベルが表示され、指定した場所へジャンプします。

⏺

カーソルベースマクロ

C-x ( で記録を開始し、C-x ) で終了、C-x e で再生します。C-u 10 C-x e で10回繰り返すことも可能です。

📋

Kill Ring サイドバー

C-c y でKill Ring履歴を一覧表示します。視覚的に選択してヤンクが可能です。履歴は永続保存されます。

🔭

アウトラインビュー

C-c o でMarkdown・HTMLの見出し階層を表示します。カーソル移動とリアルタイムに同期します（最大50,000行対応）。

💾

レジスタ サイドバー

C-c r で保存したテキストや位置を一覧表示します。視覚的に選択して利用可能です。内容は永続保存されます。

🎨

カーソル VFX

Light Particle、Cyber Trace、Kawaii Shower など9種類の視覚エフェクトを C-c 1〜9 で瞬時に切り替えます。

📎

ドキュメントインポーター

PDF・Word（.docx）・Excel（.xlsx）をプレーンテキストとして自動インポートします。50MB以下のファイルを読み取り専用で開きます。

🔍

検索・置換

インクリメンタル検索（C-s/r）、対話的置換（M-%）、ディレクトリ横断grep（M-x grep）に対応しています。

👁

ライブプレビュー

MarkdownとHTMLをリアルタイムプレビューします。C-c v でモードに応じたプレビューを自動起動し、スクロールも同期します。

🗂

ワークスペース

複数フォルダをまとめて管理します。.code-workspace 形式のファイルに対応しており、C-. で開閉します。

MCP Server

AIと直接連携。

elecxzy 自身が MCP（Model Context Protocol）サーバーとして動作します。Claude Desktop・Claude Code・Google Antigravity などのAIクライアントが、開いているバッファを読み書きし、コマンドを実行できます。

未保存の編集状態や、Piece Tableが持つ差分情報などを含めたバッファ情報をAIと共有することが可能です。

Claude Desktop Claude Code Google Antigravity
設定方法を見る →
claude_desktop_config.json
{ "mcpServers": { "elecxzy": { "command": "C:\\...\\elecxzy.exe", "args": [ "C:\\...\\app.asar" ], "env": { "ELECXZY_MCP_MODE": "true", "ELECTRON_RUN_AS_NODE": "1" } } } }
💡 M-x mcp-start でサーバーを起動、C-c m / M-x mcp-toggle で起動/停止を切り替えます。設定後はAIがバッファの内容をリアルタイムで参照できます。
